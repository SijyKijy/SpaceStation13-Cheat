# SpaceStation13-Cheat
Example internal hack for Space Station 13

# Features
* Wallhack
* Fullbright

## Key Bindings
* Insert - Unload hack
* Wallhack - Numpad 1 (On/Off)
* Fullbright - Numpad 2 (On/Off)

# Usage
1) Build dll file
2) Inject dll file into `dreamseeker` process
3) Use [key bindings](#key-bindings)
4) Profit
